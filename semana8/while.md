## While

- Lo hemos visto en DFD y en PSEINT como el ciclo mientras

- while es parecido al for pero este se basa en una condición

- cual es la sintaxis?

```
contador = 0
while condicion:
  codigo
```

```python
  contador = 1
  while edad > 18:
    contador += 1
    print(edad)
```
